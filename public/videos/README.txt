BACKGROUND VIDEO INSTRUCTIONS:

To use a background video on your website, you need to:

1. Place an MP4 format video file in this folder named "background.mp4"
2. Make sure the video is optimized for web (compressed, appropriate resolution)
3. The video should be short (10-30 seconds) and loop seamlessly
4. Keep the file size under 10MB for optimal performance

Recommended video specs:
- Resolution: 1920x1080 (HD) or 1280x720 (HD)
- Format: MP4 with H.264 encoding
- Duration: 10-30 seconds (will loop automatically)
- File size: Under 10MB

You can find free stock videos at:
- Pexels: https://www.pexels.com/videos/
- Pixabay: https://pixabay.com/videos/
- Mixkit: https://mixkit.co/free-stock-video/

After adding your video file to this folder, the website will automatically use it as a background on the landing page.
